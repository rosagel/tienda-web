-Proyecto: Tienda Web "Casa Rosagel"

- Descripción
Este proyceto se desarrolló con el uso de HTML, CSS y JavaScript. Tengo como objetivo que la gente visualice mis productos, poder agregarlos a un carrito de compras e ir calculando el total de la compra de una
forma rápida y fácil

 Funcionamiento

La página que hice muestra un catálogo de productos con su imagen, nombre y precio. Cada producto puede agregarse al carrito.

El carrito permite:

* Ver los productos agregados
* Aumentar la cantidad automáticamente (si se repiten)
* Eliminar productos
* Vaciar el carrito

El sistema calcula automáticamente:

* Subtotal
* IVA (16%)
* Total

Además, los datos del carrito se guardan en el navegador mediante LocalStorage.


- Funcionalidades que fueron implementadas

 -- Funciones principales:
